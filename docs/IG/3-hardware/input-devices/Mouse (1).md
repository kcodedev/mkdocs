## Types of computer mice
* **Mechnaical Mouse** which uses a ball and a sensor to detect how the ball rolls
* **Optical Mouse** uses a LED and sensors to detect movments
* **Laser Mouse** is a more precise version of the optical mouse
*  **Bluetooth Mouse** Uses Bluetooth or RF signals
