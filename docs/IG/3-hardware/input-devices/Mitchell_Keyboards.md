# Role of Keyboards


The role of a keyboard is to serve as an input device for users to enter text characters into a computer, and to interact with the computer.


# Components


- **Keycaps** - Provide the surface used to press down the Switches
- **Switches** - Register the press of a key
- **Plate** - Holds switches in place
- **PCB** - Sends the signal received from a keypress to the computer
- **Case** - Holds together the other components


# Different Keys


1. **Arrow Keys**
    * Left
    * Right
    * Up
    * Down
2. **Navigational Keys**
    * Insert
    * Delete
    * End
3. **Function Keys**
    * F1
    * F8
    * F12
4. **Number Keypad** (Numpad) - Used to make work with lots of numbers easier and quicker
    * Numeric Keys
    * Arithmetic operators 
5. **Alphanumeric Keys**
    * Letters
    * Numbers

_(Some keyboards drop certain features, and/or make the keys more compact)_

# Different types


_Mechanical keyboards_ use individual switches for every key. They are very durable, often lasting for tens of millions of keypresses. They have very good responsiveness, and allow for a greater selection of sounds that your keys can make when pressed (Tactile, Linear, Quiet, Clicky, Creamy)

# To Know

- Keyboards send data to computers via Binary. This is then turned into characters through sets like ASCII.
- Keyboards often use a QWERTY layout in English.
- Cons of keyboards are that they take a long time to learn how to use, and that they are easy to make typing errors on.
- Keyboards use Serial Transmission.
