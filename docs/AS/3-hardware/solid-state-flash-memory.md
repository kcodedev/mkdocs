# Solid State (Flash) Memory

![ssd](https://upload.wikimedia.org/wikipedia/commons/thumb/7/75/Samsung_980_PRO_PCIe_4.0_NVMe_SSD_1TB-top_PNr%C2%B00915.jpg/960px-Samsung_980_PRO_PCIe_4.0_NVMe_SSD_1TB-top_PNr%C2%B00915.jpg)

- **Operation**: Stores data in NAND or NOR flash memory chips using electrical charges in floating-gate transistors.
- **Key Processes**: Programming (writing), reading, and erasing data blocks; no moving parts, relying on electronic switching.
